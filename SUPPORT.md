# Support

## Getting Help

Use GitHub Issues for:

- bugs
- rendering problems
- install issues
- docs-check false positives or missing checks
- feature requests

Before opening an issue, please try:

```sh
shine version
shine --check README.md
shine --plain README.md
```

If you built from source, use:

```sh
bin/shine version
bin/shine --check README.md
```

## What To Include

- operating system
- terminal app
- install method
- `shine version`
- command you ran
- minimal Markdown input when possible
